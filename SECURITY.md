# Security Policy

**PLEASE DON'T DISCLOSE SECURITY-RELATED ISSUES PUBLICLY, [SEE BELOW](#reporting-a-vulnerability).**

## Supported Versions

Security fixes are provided for the currently maintained version of this project, package, application, or service.

## Reporting a Vulnerability

If you discover a security vulnerability within EmbegeQ, please send an email to the security team at **security@embegeq.dev.dyzulk.com**. All security vulnerabilities will be promptly addressed. We appreciate your efforts to responsibly disclose your findings.
